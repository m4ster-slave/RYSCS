# RYSCS

**RPGYUMYUMS SUPER COOL SHELL**

- RYSCS is a simple crossplatform shell with basic features might add some cool stuff we will see how much fun I have working on this.
    - cd
    - help
    - exit
    - ls
    - pwd
    - rm
    - mv
    - cp
    - mkdir
    - rmdir
    - cat
    - touch
    - chmod
    - clear
