## EXTRA BUILTINS 
- **grep**: Search for patterns in files. 
- **ps**: Report a snapshot of the current processes. 
- **kill**: Terminate processes by ID or name. 
- **wc**: Count lines, words, and characters in files.

## How to run
To run the shell right now, 
    - clone the repo
    - run ```make```
    - run ```./RYSCS```
